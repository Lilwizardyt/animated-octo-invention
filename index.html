Project on console

Microsoft Visual Studio is an integrated development environment (IDE) from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps. ... Visual Studio includes a code editor supporting IntelliSense (the code completion component) as well as code refactoring.

Step 1 - Make sure your computer is ready for Visual Studio. Step 2 - Download Visual Studio. Step 3 - Install the Visual Studio installer. Step 4 - Choose workloads.

How to open console

you can usually do CTRL-ALT-O to make it visible. Or through menus using View->Output. You can run your program by: Debug -> Start Without Debugging . It will keep a console opened after the program will be finished.

A console window opens and closes immediately. You are not even able to read the Hello World. In e.g. Visual Studio Code (or other IDEs like e.g. Eclipse) console applications are running on a console integrated in the IDE. When the app has finished, the output is still visible.
